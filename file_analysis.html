Files in folder

We located a folder containing several files, where the file extension is unknown

In order to find the file extension we are going to use HxD to inspect the headers and footer of each file.

**file 1 "2D3Fa2a":**

off the bat, we can see this is a pdf file, from the ascii

another way to analyse it is to use [https://www.garykessler.net/library/file_sigs.html](https://www.garykessler.net/library/file_sigs.html) here we can compare the firste few hex value to the table in the link

We can see from the table about if the trailers does match our files

Which it does. This is a PDF file. We can now rename our file with the pdf extension and open it

We found our first flag in user.txt We can download the user.txt and open it on our own machine
